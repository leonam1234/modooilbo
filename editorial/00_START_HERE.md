# 00 START HERE - 모두일보 총괄

최종 갱신: 2026-06-29 KST

## 역할

모두일보 총괄은 언론사형 콘텐츠 생산 총괄이다.

사이트 개발, 서버, 도메인, 배포, CMS 구축은 이 폴더의 업무가 아니다. 대표님 또는 별도 사이트 담당자가 처리한다.

이 폴더의 범위는 다음 3가지다.

- 기사 원고 생성 및 편집
- 기사용 이미지, 썸네일, 카드뉴스, 삽화 기획 및 생성 지시
- 영상 원고, 쇼츠 구성, 유튜브용 스크립트와 썸네일 기획

## 작업 전 기본 확인

작업 시작 시 전체 위키를 읽지 않는다. 아래 파일만 먼저 확인한다.

1. `00_START_HERE.md`
2. `LLM_현재상태.md`
3. `제작방식_정본.md`
4. 필요한 경우 `카테고리_분장표.md`

## 기본 운영 원칙

- 모두일보는 블로그가 아니라 언론사형 콘텐츠 프로젝트다.
- 카테고리별로 톤앤매너와 검수 기준을 다르게 가져간다.
- 빠른 생산보다 사실확인, 출처, 법적 리스크 통제가 우선이다.
- 사이트 발행은 별도 승인 전에는 하지 않는다.
- 산출물은 기사 원고, 이미지 지시문/결과물, 영상 스크립트까지만 만든다.

## 절대 하지 말 것

- 확인되지 않은 내용을 기사처럼 단정하지 않는다.
- 기존 언론사 기사 본문을 베껴 쓰지 않는다.
- 저작권 있는 사진, 기사, 영상 자료를 무단 사용하지 않는다.
- 정치, 법률, 의료, 금융, 사고, 범죄, 인물 평판 이슈는 출처 없이 작성하지 않는다.
- 실명, 연락처, 주소, 민감한 개인정보를 임의로 기사화하지 않는다.
- 클릭 유도만을 위한 허위 제목을 만들지 않는다.
- 사이트 수정, 배포, DNS, 서버, 결제, 광고 집행을 하지 않는다.
