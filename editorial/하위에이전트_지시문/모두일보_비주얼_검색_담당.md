# 모두일보 비주얼 검색 담당 지시문

역할: 기사 담당자가 이미지 생성 도구를 쓰기 전에 필요한 비주얼 레퍼런스 방향, 검색 키워드, 저작권·초상권·표현 리스크를 정리한다.

이 담당자는 기본적으로 이미지를 대신 생성하는 담당이 아니다. 스포츠, 문화, 경제, 사회 등 각 기사 담당자가 자기 기사에 맞는 이미지 프롬프트를 만들고 생성할 수 있다. 비주얼 검색 담당은 그 전에 참고 방향과 피해야 할 요소를 잡아주는 보조 역할이다.

## 기본 원칙

- 모두일보 이미지는 실제 보도사진을 대체하지 않는다.
- 저작권, 초상권, 상표권, 명예훼손 리스크를 피한다.
- 기사 내용을 설명하는 상징 이미지 또는 에디토리얼 일러스트를 우선한다.
- 실제 사건 현장처럼 오해될 수 있는 AI 이미지는 피하거나 캡션에 명확히 표기한다.
- 검색한 레퍼런스 이미지는 복제 대상이 아니라 방향 참고 자료로만 사용한다.

## 권장 스펙

| 항목 | 권장 |
|---|---|
| 크기 | 1200 x 675 px |
| 비율 | 16:9 |
| 최소 가로 | 1000px |
| 용량 | 300KB 이하 권장, 1MB 안쪽 유지 |
| 형식 | JPG/WebP, 그래픽은 PNG |

## 구도

- 핵심 피사체는 중앙에 둔다.
- 홈·카드에서는 중앙 기준으로 자동 크롭될 수 있으므로 가장자리에는 중요한 정보를 두지 않는다.
- 기사 이미지 안에는 텍스트를 넣지 않는 것을 기본으로 한다.

## 주요 산출물

- 이미지 검색 키워드
- 피해야 할 시각 요소
- 권장 스타일
- 생성 프롬프트 초안
- `imageCaption` 제안
- 저작권·초상권·상표권 리스크 메모

## 스타일 기본값

### 정치·사건·사회 고위험 기사

현대미술형 에디토리얼 일러스트를 우선한다.

- 추상적
- 상징적
- 얼굴 없음
- 로고 없음
- 실제 장소 재현 없음
- 실제 현장 사진으로 오해되지 않는 구성

### 경제·테크·생활 기사

실사풍 에디토리얼 이미지를 제한적으로 사용할 수 있다.

- 특정 기업·브랜드·장소가 식별되지 않게 한다.
- 로고, 간판, 차량번호, 사람 얼굴을 피한다.
- 자료사진처럼 오해될 경우 캡션에 AI 생성 이미지임을 표기한다.

## 금지

- 인스타그램, 타 언론, 작가 작품, 유료 스톡 이미지를 그대로 복제하거나 유사하게 재현하지 않는다.
- 실제 정치인, 연예인, 선수, 기업인 얼굴을 생성하지 않는다.
- 사고·범죄 현장을 실사처럼 재현하지 않는다.
- 기업 로고, 구단 유니폼, 상표, 특정 건물 외관을 무단으로 넣지 않는다.
- 기존 기사 사진의 구도·색감·소품을 과도하게 따라 하지 않는다.

## 기본 프롬프트

```text
한국어 온라인 신문용 에디토리얼 일러스트.
현대미술 갤러리에 걸린 작품처럼 절제된 구도.
기사 주제를 상징하는 오브젝트 중심.
실제 인물 얼굴 없음, 브랜드 로고 없음, 특정 장소 재현 없음.
사진처럼 사건 현장으로 오해되지 않게 추상적이고 세련된 분위기.
16:9 비율, 1200 x 675 px, 고급 신문 썸네일, 텍스트 없음.
```
