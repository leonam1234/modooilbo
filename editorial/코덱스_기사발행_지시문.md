## 기사 = 파일 1개
`content/articles/<슬러그>.md` 로 **1기사 1파일**. 슬러그 = 영문 소문자·숫자·하이픈, **날짜 접두사로 고유하게**(예: `2026-06-30-budget-clash`).

형식:
```markdown
---
title: 기사 제목
category: politics          # 아래 8개 중 하나(영문)
author: 김도윤 / 정치부 기자
publishedAt: 2026-06-30 14:00   # KST(한국시각)
tags: 추경, 국회
summary: 한 줄 요약
---
본문 첫 문단.

본문 둘째 문단. (문단은 빈 줄로 구분 → 자동으로 <p>)
```

## 발행 흐름
1. `content/articles/`에 `.md` 작성
2. `npm run content` 실행(데이터 변환 + 이미지 자동) → `npm run build`로 green 확인
3. **전용 브랜치**에서 `git commit` (`master` 직접 커밋 금지). 커밋 직전 `git status`로 혼입 확인.
4. **배포는 하지 말 것.** `npm run deploy:prod`는 사람 승인 후. → 커밋까지만 하고 **"게시 대기 — 배포 승인 요청"**으로 보고하고 멈춘다.
